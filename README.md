# Movie-Recommendation-System
A simple Movie Recommendation System built using Python that recommends movies based on genres and user ratings.

Features
Content-Based Filtering
Collaborative Filtering
Movie Similarity Recommendations
User-Based Recommendations
Technologies Used
Python
Pandas
Scikit-learn
NumPy
Installation
Clone the repository: git clone https:Movie_Recommendation_System.ipynb

Install dependencies: pip install pandas scikit-learn numpy

Run the project: python movie_recommender.py
